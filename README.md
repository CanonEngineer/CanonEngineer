## Sobre

Construo software ponta a ponta — da infraestrutura de rede e segurança até interfaces web, IoT e desktop.  
O hub visual de todo o código fica na **[Tree of Knowledge](https://canonengineer.github.io/TreeofKnowledge/)**: cada projeto vira uma árvore com nós clicáveis, hover explicativo e página de implementação.

| Foco | O que entrego |
|------|----------------|
| **Mapa de código** | Árvore interativa de todos os repositórios |
| **Redes & segurança** | CCNA, CyberOps, análise defensiva |
| **Web & APIs** | JavaScript, PHP, Django, REST |
| **IoT & automação** | Smart Home, telemetria, sensores |
| **Desktop corporativo** | Customizações Qt / Veyon (HCFMB) |
